# electron-ebook-sync
## prerequisiti
* [NodeJS](https://nodejs.org/en/download/current/)
* Electron `yarn global add electron`  OR `npm i -g electron`
## installazione
### client
* `cd client`
* `yarn` OR `npm install`
### server
* `cd server`
* `yarn` OR `npm install`

## utilizzo
### client
* `electron ./client/`

### server
* `node ./server/main.js`
