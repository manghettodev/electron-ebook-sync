# electron-ebook-sync
## dependencies
* [NodeJS](https://nodejs.org/en/download/current/)

## setup
### client
* `cd client`
* `yarn` OR `npm install`
### server
* `cd server`
* `yarn` OR `npm install`

## start
### client
* `cd client`
* `yarn electron .` OR (if you installed electron globally) `electron ./client/`

### server
* `node ./server/main.js`
